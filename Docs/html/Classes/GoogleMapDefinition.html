Overview

Helper class used to define a map to be opened in Google Maps. Note that there’s a good chance some of these properties will be nil, but either the queryString or the center property should be set.

Properties

center

Location in lat/long. If both this and the queryString are specified, this will be used as a center for the search. To clear this value, set it to kCLLocationCoordinate2DInvalid.

@property (nonatomic, assign) CLLocationCoordinate2D center
